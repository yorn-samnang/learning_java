Java String Methods (Quick Guide)

charAt(i) -> Get character at index
Example: "Hi".charAt(1) -> i

substring(a, b) -> Get part of string
Example: "Hello".substring(1, 4) -> ell

join("-", a, b, c) -> Join strings with separator
Example: String.join("-", "A", "B") -> A-B

concat(str) -> Add another string
Example: "Hi".concat("!") -> Hi!

replace('a', 'o') -> Replace characters
Example: "banana".replace('a', 'o') -> bonono

replaceFirst("x", "y") -> Replace first match
Example: "xxX".replaceFirst("x", "y") -> yxX

replaceAll("x", "y") -> Replace all matches
Example: "xxX".replaceAll("x", "y") -> yyX

repeat(n) -> Repeat string
Example: "Hi ".repeat(3) -> Hi Hi Hi 

indent(n) -> Add spaces to each line
Example: "Hi".indent(4) -> "    Hi"
